
# App Engagement Analytics Project (BI Portfolio)

This project demonstrates SQL, Python, and data analysis skills through a simulated dataset of mobile app session events.

## Files Included
- session_data.csv (500 rows of simulated user sessions)
- analysis_queries.sql (SQL queries to analyse engagement)
- analysis.py (Python script using Pandas)
- README.md (project documentation)

## Metrics Covered
- Daily Active Users (DAU)
- Average Session Length
- Country & Device Engagement Patterns

Use this project as a BI example for portfolio or applications.
